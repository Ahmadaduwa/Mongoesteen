# Mongoesteen Project

This project focuses on data processing and model training for predicting the volume of mangosteen fruit using machine learning techniques.

## Project Structure

- **ModelTraining.py**: Script for training the machine learning model.
- **PrepareDatasetForRegression.py**: Prepares datasets for regression tasks.
- **train.ipynb**, **own_train.ipynb**: Jupyter notebooks for model training.
- **test.ipynb**, **own_test.ipynb**: Jupyter notebooks for model evaluation.
- **DataMangosteen**, **Datasets**, **DataTest**, **DataTrain**: Folders containing images, labels, and related data.

## How to Use

1. Prepare your data in the specified folders.
2. Run `PrepareDatasetForRegression.py` to preprocess the data for regression.
3. Train the model using `ModelTraining.py` or the provided Jupyter notebooks.
4. Evaluate the model using the test notebooks.

## Requirements

- Python 3.x
- TensorFlow
- Keras
- pandas
- numpy
